바닐라 자바스크립트 공부
==================

목표
---
- 강의, 혹은 구글링으로 학습한 지식을 보지만 말고 직접 코드를 작성하고 익히는 것!
- 라이브러리, 프레임워크에 의존하지 않고 순수 자바스크립트를 배우는 것!
- 자바스크립트의 동작 원리를 아는 것!

## 규칙
* Commit 기준
 * 학습한 내용에 따라 commit할 것!
* 폴더구조
 * 학습 주제에 따라 세분화 할 것, 공통 주제는 폴더로 만들 것!
* 많이가 아니라 오래 꾸준히 할 것!
* 공부가 목적이므로 지나치게 정리하는 것에 빠지지 말 것!
